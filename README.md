##  Project Overview
This project presents an interactive **Banking Churn Analytics Dashboard** built using Power BI to analyze customer behavior, identify churn patterns, and support data-driven marketing decisions.
## Objectives
* Analyze customer churn trends
* Identify high-risk customer segments
* Improve customer retention strategies
* Enable business decision-making using data insights
## Tech Stack
* Power BI (Dashboard & Visualization)
* SQL (Amazon Redshift style queries)
* Data Modeling & DAX
* Excel/CSV Data Source
## Dataset
* Customer demographics (Age, Gender, Geography)
* Account details (Balance, Products)
* Churn Indicator (Exited)
## Key Features
* Interactive KPI Dashboard (Total Customers, Churn Rate, Retention Rate)
* Customer Segmentation (Age Group, Geography, Products)
* Drill-through for detailed customer analysis
* Dynamic filters (Slicers) for real-time insights
* Advanced DAX measures for business KPIs
##  Business Impact
* Helps marketing teams target high-risk customers
* Improves retention strategy planning
* Enables data-driven decision-making
## How to Use
1. Download the `.pbix` file
2. Open in Power BI Desktop
3. Interact with filters and visuals
